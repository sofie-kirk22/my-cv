# Personal Resume Website

This repository contains the source code for my personal resume and portfolio website.  
The site presents my professional background, technical skills, projects, and selected academic work from my time at Aarhus University.

The goal of the project was both to create a clean, accessible portfolio and to further develop my skills in modern frontend development using **TypeScript, React, and Next.js**.

---

## Live Website

You can view the deployed website here:

**Portfolio Website:**  
https://sofie-kirk22.github.io/my-cv/

The site includes:

- Professional profile
- Technical skills overview
- Project portfolio
- Academic research work
- Resume download
- Contact links

---

## Features

- Responsive design for desktop and mobile
- Dark mode and light mode support
- Modular component architecture
- Interactive project cards
- Embedded PDFs for academic work
- Expandable tag lists for technologies
- Smooth navigation between sections

## Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/your-repository-name.git
```

Navigate to the project directory:

```bash
cd your-repository-name
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the site in your browser:

```
http://localhost:3000
```

---

## Key Design Decisions

The site was designed with the following principles:

- **Clarity** – projects and experience should be easy to scan quickly
- **Modularity** – reusable React components for maintainability
- **Accessibility** – readable typography and semantic structure
- **Performance** – optimized images and minimal client-side complexity
